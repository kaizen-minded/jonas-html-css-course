The Basic Language of the Web: HTML

Posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three Fundamentaltechnologies: HTML, CSS, Javascript. These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn.

What is HTML?

HTML stands for Hyper Text Markup Language. It's a markuup language that web developers use to structure the content of a webpage (not a programming language).

HTML consists of elements that describe different tupes of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts

  1. The opening tag
  2. The content
  3. The closing tag

You can learn more at MDN Docs

Why should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Worpress or Wix
  • To build web applications
  • To impress friends
